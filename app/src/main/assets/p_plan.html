Plan Page

This page has a few items to use for flight planning.

  • Web Page Buttons - opens the associated page in a web browser. Requires Internet access as indicated by status.
  • Offline Flight Plan Form - displays an ICAO flight plan form that can be filled out for reading to a briefer over telephone or radio.
  • Pretend - sets the position of the airplane at the given lat/lon. It can also pretend to be flying at the given speed, heading, altitude and turn rate. The (capture center) button copies the current lat/lon of the center of the chart page to the lat/lon boxes.
  • Find airports - generates a list of airports within a certain distance and heading from another airport. Useful for looking for airports to make a fuel stop at.
