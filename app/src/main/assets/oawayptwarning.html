The OA database is a community maintained airport and navaid database. It is derived from data obtained from OurAirports.

The OA database is not authoritative. Neither OuterWorldApps nor OurAirports warrant that any information contained in the database is accurate or useful for any purpose. Always be extremely skeptical regarding any information provided by the database.

  • You MUST assume it contains errors, including missing or out-of-date information.
  • You MUST verify all information with official sources before using it for any flight related purpose.
  • You MUST use current, authoritative, official sources as your primary source of information before and during flight.

The indication of valid to simply indicates that the information was obtained from OurAirports within 56 days prior to that date. It does not indicate that it has been verified in any way or is accurate to any extent.

Read the Site Info pages linked at the top of the OurAirports page. Any errors discovered in the database should be reported to Contact under the Site Info links.
