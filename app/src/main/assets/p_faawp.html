FAAWP1 and FAAWP2 pages

These pages are used to view information related to FAA-defined waypoints. They consist of airports, fixes, localizers and navaids. Each page is independent and can be used to select different waypoints.

  • Enter the identifier of the desired fix in the text box at the top and click the Find button. You may also enter search terms such as a city name. Entering multiple words requires that the waypoint description contains all the words given in the search box.
  • You may also click one of the history buttons listed below the text box to get a recent fix. To delete a button from this list, do a long-click on the button.
  • The GoTo button will set the waypoint as the current destination and a line will be drawn from the airplane to the destination location on the Chart page.
  • The Locate button will center the moving map of the Chart page on the waypoint.
  • If it is an airport, the METAR button will link to the FAA's METAR page. Internet access is required for this to work.
  • If it is an airport, the Info button will display AFD-like information for the airport. The plates for the airport's state must be downloaded for this button to work.
  • Summary information for the waypoint will be displayed next.
  • Finally, for airports, provided the plates for the airport's state have been downloaded, buttons will be shown to display plates for this airport, including:
    • FAA's official airport diagram if any
    • Any SIDs and STARs
    • Instrument approaches
    • Alternate and takeoff minimums
    • WairToNow runway diagram
    Clicking on the button will display the item. To return to the button page, either click the Android's Back button, or click the FAAWP1 or FAAWP2 button.
