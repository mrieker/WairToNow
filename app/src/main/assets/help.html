WairToNow, version

... an aviation oriented map program from Outer World Apps.

Site Map  

The tab-like buttons at the bottom of the screen select the page to display:

  • Chart - main moving-map page
  • Waypt1 - search for database-defined waypoint info (including airport info, runway diagrams, approach plates, etc)
  • Waypt2 - same as Waypt1
  • UserWP - define and search for user-defined waypoint
  • Glass - glass cockpit style display
  • Route - manage route of flight
  • Crumbs - record/display a breadcrumb trail
  • Plan - some flight planning tools
  • VirtNav1,2 - virtual nav displays
  • Options - option settings
  • Maint - chart and data downloading
  • Sensors - sensor selection
  • Files - data file explorer

NOTE: The buttons scroll horizontally in a detent fashion. You must swipe at least a fifth of the way across the screen to scroll. This holds the buttons steady in a bumpy cockpit.

Menu in upper-right corner

  • Chart - alternate way to get to chart page
  • Exit - closes the app
  • Re-center - puts airplane at center of screen, moving chart as needed. Chart will then move as needed to maintain aircraft at center until display is panned or zoomed.
  • Hide Tabs - hides the tab buttons at bottom of screen to save space
  • << MORE - pops up a temporary menu with the hidden tab buttons
  • Lock Screen - locks the screen in whatever orientation it is in
  • Unlock Screen - unlocks the screen allowing user to rotate display

Downloading charts

You must download data before using the app. Download by clicking the Maint button.

  • You must download the waypoints from the Misc button. This waypoint data includes airports, fixes, localizers, navaids and runways for the entire country.
  • Everything else is optional, select what you want. Avoid over-selecting as the download times and flash usage can be significant.

WairToNow has access to an unofficial database and charts obtained from OpenFlightMaps covering parts of Europe. Please read their General User's License as linked at the bottom of that page before using any of the OFM charts and/or waypoint database. Also, please support their effort by reporting any inaccuracies you find in the OFM charts and/or waypoint database to info@openflightmaps.org Must be enabled on the Options page.

WairToNow has access to database obtained from ourairports.com which has unofficial waypoints around the world. There are no charts in this data source. Please visit their site, click on Site info then About to read licensing information. And support their effort by reporting any inaccuracies you find the the database to contact@ourairports.com Must be enabled on the Options page.
