WairToNow, version

... an aviation oriented map program from Outer World Apps.

The menu buttons at the bottom select the basic screen:

  • Chart - main moving-map display using the selected chart as background
  • FAAWP1 - search for FAA-defined waypoint and display information about it
  • FAAWP2 - same as FAAWP1
  • UserWP - search for user-defined waypoint
  • Glass - glass cockpit style display
  • Crumbs - record/display a breadcrumb trail
  • Plan - some flight planning tools
  • Options - option settings
  • Maint - chart and data downloading
  • GPS - GPS receiver status
  • Files - data file explorer

Downloading charts

You must download data before using the app. Download by clicking Maint button.

NOTE: If you download plates, it will give you the airport diagrams, SIDs STARs, IAPs, etc. It will also give you a runway diagram that is synthesized by WairToNow as many airports do not have an official FAA airport diagram. These are constructed from the FAA data that has the lat/lon of most but not necessarily all runways for an airport. Some airports don't have any runway lat/lon information provided at all, so you may not see some or any runways for a given airport.

That being said, WairToNow will use OpenStreetMap tiles as a background for the diagram which most times has runway and taxiway outlines which are generally useful. When you download the plates, WairToNow will start downloading the tiles for the corresponding runway backgrounds in a background thread. The app will still function while this downloading is happening.

  • You must download the waypoints from the Misc button. This waypoint data includes airports, fixes, localizers, navaids and runways for the entire country.
  • Everything else is optional, select what you want. Avoid over-selecting as the download times can be significant.
  • NOTE: There is no selection for the Street maps, they are downloaded as needed, which requires an Internet connection at the time of use. You can preload areas of interest by previewing the area where you have an internet connection, and WairToNow will save the tiles, so in that case an Internet connection would not be needed to view those same areas again. Be sure to use a wide range of zoom when previewing so it will load all the tiles for the different zoom levels.
